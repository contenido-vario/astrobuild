Páginas y enrutamiento

Las páginas en Astro utilizan un sistema conocido como file based routing o enrutamiento basado en archivos. Para crear una nueva página en Astro se crea un nuevo archivo en la carpeta pages, y se les da el nombre que se desee en minúsculas y con extensión .astro. Creamos por ejemplo una página about.astro en la misma carpeta donde se encuentre index.astro. Si tú quieres crear tu propia página de 404 deberías crear un nuevo archivo y llamarlo 404.astro y luego crear tu página 404 personalizada, la cual incluso puede utilizar un Layout.
